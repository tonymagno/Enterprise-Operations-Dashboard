# Enterprise Operations Dashboard

Plataforma profissional de gestão e telemetria com foco em operação, indicadores, alertas e análise executiva.

## Stack
- Next.js
- TypeScript
- FastAPI
- PostgreSQL
- Docker
- GitHub Actions
- AWS

## Objetivo
Criar um dashboard premium para monitorar KPIs, eventos, alertas, logs e relatórios em tempo real ou quase em tempo real.

## Estrutura
- `apps/web` → Frontend
- `apps/api` → Backend
- `docs` → Arquitetura e roadmap
- `infra` → Infraestrutura
- `database` → Migrações e seeds
- `packages` → Código compartilhado
